## Project Overview

**Name:** DanteCode (`dantecode`)
**Version:** 1.0.0
**License:** MIT
**Type:** Model-agnostic AI coding agent — VSCode extension + CLI + monorepo SDK
**Repository:** github.com/dantericardo88/dantecode
**Branch:** `dad-ready`

DanteCode is a portable, model-agnostic coding agent with a DanteForge verification layer. It competes directly against Cursor, GitHub Copilot, Windsurf, Augment, Claude Code, Cline, Aider, Continue.dev, and all major AI coding tools. The goal is to reach 9+ on every competitive dimension in a 28-dim scoring matrix against 15 closed-source and 13 OSS competitors.
