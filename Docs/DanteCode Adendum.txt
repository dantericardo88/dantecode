10.1 Grok Prioritization (Update to D2 Architecture & Tech Stack)

Default Provider: Explicitly set Grok API (x-ai/grok-4.20-beta-latest-non-reasoning for parsing tasks, grok-4-1-fast-reasoning for complex logic) as the default model for all operations. Use environment variable GROK_API_KEY for authentication.
Fallback Logic: If Grok fails (e.g., rate limit, downtime), prompt the user for a Claude key and switch to Anthropic Claude Sonnet 4.6 as fallback. Add to D3 Core Features: "Model switching command: dantecode model switch claude with automatic fallback if Grok is unavailable."
