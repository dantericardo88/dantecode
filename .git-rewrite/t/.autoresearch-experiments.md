# Overnight Autoresearch Experiments

## Experiment Queue (Run Sequentially)

### 1. Agent Loop Performance (3 hours)
**Goal:** Reduce time to first token in agent responses  
**Metric:** Milliseconds from prompt submission to first response chunk  
**Baseline:** Run simple "Hello" prompt, measure latency  
**Approach:** Try lazy loading, caching, async optimizations  
**Time Budget:** 3 hours  

**Measurement Command:**
```bash
time node --eval "
const { runAgentLoop } = require('./packages/cli/dist/index.js');
const start = Date.now();
runAgentLoop({ prompt: 'Hello', onFirstChunk: () => {
  console.log(Date.now() - start);
  process.exit(0);
}});
" 2>&1 | grep -oP '\d+' | head -1
```

---

### 2. Test Suite Speed (2 hours)
**Goal:** Reduce test execution time  
**Metric:** Total seconds for `npm test --workspace=packages/core`  
**Baseline:** Current runtime (measure first)  
**Approach:** Parallel test execution, mock optimization, test isolation  
**Time Budget:** 2 hours  

**Measurement Command:**
```bash
time npm test --workspace=packages/core 2>&1 | grep "Duration" | grep -oP '\d+ms' | sed 's/ms//' || echo "0"
```

---

### 3. CLI Bundle Size (2 hours)
**Goal:** Reduce packages/cli/dist size  
**Metric:** Total bytes in dist/index.js  
**Baseline:** Current 15MB  
**Approach:** Dynamic imports, tree shaking, dependency elimination  
**Time Budget:** 2 hours  

**Measurement Command:**
```bash
ls -l packages/cli/dist/index.js | awk '{print $5}'
```

---

### 4. Memory Efficiency (2 hours)
**Goal:** Reduce peak memory usage during agent loop  
**Metric:** Peak RSS (resident set size) in bytes  
**Baseline:** Run typical session, measure peak  
**Approach:** Buffer pooling, streaming optimization, cache limits  
**Time Budget:** 2 hours  

**Measurement Command:**
```bash
node --expose-gc packages/cli/dist/index.js --eval "
const used = process.memoryUsage().rss;
console.log(used);
" 2>&1 | tail -1
```
